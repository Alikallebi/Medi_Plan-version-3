# Scénarios de démonstration

## Scénario 1 : Validation complète
1. Se connecter en tant que Chef de Service.
2. Créer un planning (Module 2).
3. Soumettre pour validation.
4. Basculer sur validateur RH.
5. Approuver.
6. Basculer sur Super Admin.
7. Validation finale.

## Scénario 2 : Gestion de conflit
1. Soumettre un planning avec problème.
2. Rejeter avec motif explicite.
3. Modifier puis re-soumettre.
4. Vérifier l'historique et la nouvelle validation.

## Scénario 3 : Administration
1. Ouvrir le dashboard admin workflow.
2. Identifier un planning bloqué.
3. Relancer un validateur.
4. Consulter l'audit trail.
5. Exporter les données.
